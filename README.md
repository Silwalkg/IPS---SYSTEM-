# AI-Based Intrusion Prevention System (IPS)

An intelligent IPS system using Machine Learning for network traffic analysis and threat detection.

## Features
- Real-time network traffic monitoring
- ML-based anomaly detection
- Threat classification
- Automated response mechanisms

## Structure
- `src/` - Core IPS components
- `models/` - ML model definitions and trained models
- `data/` - Training data and logs
- `config/` - Configuration files

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
python src/main.py
```
